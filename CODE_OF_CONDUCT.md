# Contributor Covenant Code of Conduct

## Our Standards
Examples of behavior that contributes to a positive environment:
- Using welcoming and inclusive language
- Being respectful of different viewpoints
- Accepting constructive criticism
- Focusing on what is best for the community

Examples of unacceptable behavior include:
- Harassment, trolling, or personal attacks
- Publishing others' private information
- Conduct inappropriate for a professional environment

## Enforcement Responsibilities
Project maintainers are responsible for clarifying acceptable behavior and taking corrective action when needed.

## Scope
This Code applies both inside project spaces and in public spaces where individuals represent the project.
